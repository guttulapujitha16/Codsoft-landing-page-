🍦 Ice Cream Shop Landing Page

A beautiful and responsive Ice Cream Shop Landing Page built using HTML and CSS. This project showcases a modern UI design with attractive layouts, smooth styling, and a user-friendly interface perfect for promoting ice cream brands, dessert shops, or café businesses.

✨ Features

- Responsive landing page design
- Modern and colorful UI
- Smooth hover effects and animations
- Hero section with attractive banner
- Product showcase section
- Customer review/testimonial section
- Contact and footer section
- Pure HTML & CSS implementation

🛠️ Technologies Used

- HTML5
- CSS3

📌 Purpose

This project was created to practice frontend web development skills and build an aesthetically pleasing landing page using only HTML and CSS.

🚀 Getting Started

1. Clone the repository
2. Open the project folder
3. Run "index.html" in your browser

A stylish and delicious-looking landing page designed for ice cream lovers.
